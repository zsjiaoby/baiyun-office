# 📖 白云办公室预约表单使用说明

---

## 🎯 功能流程

```
微信/短信发链接 → 客户填写表单 → 提交后 → 飞书/企业微信群自动收到提醒
```

---

## 🚀 第一步：配置飞书机器人

### 1. 创建飞书机器人
1. 打开飞书群 → 右上角「...」→ 群设置 → 机器人
2. 点击「添加机器人」→ 选择「自定义机器人」
3. 输入名字「办公室预约提醒」，选择头像，点击「添加」
4. ✅ 重要：复制这个 webhook 地址，大概长这样：
   ```
   https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxx
   ```

### 2. 配置到代码中
打开 `server.js`，把第 11 行的：
```javascript
const FEISHU_WEBHOOK = 'https://open.feishu.cn/open-apis/bot/v2/hook/你的飞书机器人key';
```
替换成你刚才复制的真实地址。

---

## ☁️ 第二步：部署上线（获得可分享的链接）

### 🆓 免费方案：Vercel 一键部署（推荐）
1. 把整个 baiyun-office 文件夹上传到 GitHub
2. 打开 https://vercel.com/new
3. Import 你的仓库，直接点 Deploy
4. ✅ 部署成功，获得类似这样的链接：
   ```
   https://baiyun-office.你的用户名.vercel.app
   ```

### 🆓 免费方案：Netlify 部署
1. 打开 https://app.netlify.com
2. 直接把整个文件夹拖进去
3. 等待 30 秒，获得链接

### 💳 付费方案：云服务器
1. 阿里云/腾讯云买个轻量服务器（50元/月）
2. 上传代码，执行：
   ```bash
   npm install
   npm start
   ```
3. 用你的域名访问

---

## 📱 第三步：发给客户

### 微信分享
1. 把部署好的链接发给自己
2. 在微信里打开，点击右上角「...」→ 「发送给朋友」
3. ✅ 客户在微信里可以直接点开填写

### 短信发送
直接发：
```
您好！这是我们办公室需求调研表，请您填写一下，我们将尽快为您匹配合适房源：
https://你的链接.com
```

## 📊 进阶：数据自动存入飞书多维表格

### 1. 创建多维表格
1. 飞书新建多维表格，列名：
   - A列：姓名
   - B列：电话
   - C列：需求面积
   - D列：工位数量
   - E列：隔断数量
   - F列：意向区域
   - G列：提交时间
