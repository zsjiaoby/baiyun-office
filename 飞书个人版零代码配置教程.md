## 🚀 配置步骤（5分钟搞定）

---

### 第一步：创建只有你一个人的飞书群

对，就是自己和自己聊天！

1. 飞书APP → 右上角 `+` → 创建群组
2. 名字：「客户线索」
3. 直接创建，不需要加别人

---

### 第二步：添加群机器人

1. 群设置 → 机器人 → 添加机器人
2. 选择「自定义机器人」
3. 名字：「新客户提醒」，选个头像
4. ✅ 复制这个 webhook 地址
5. 粘贴到 `server-feishu-only.js` 第14行

---

### 第三步：创建多维表格（用来存客户）

1. 飞书新建 **多维表格**，名字：「客户线索库」
2. 添加这些列（字段类型按下面选）：

| 列名 | 字段类型 |
|------|----------|
| 姓名 | 单行文本 |
| 电话 | 单行文本 |
| 需求面积 | 单行文本 |
| 工位数量 | 单行文本 |
| 隔断数量 | 单行文本 |
| 换房原因 | 单选 |
| 意向区域 | 单行文本 |
| 提交时间 | 日期时间 |
| 跟进状态 | 单选 → 待联系/已联系/已带看/已成交/已放弃 |

3. ✅ 把跟进状态默认值设为「待联系」

---

### 🎯 最神奇的一步：飞书捷径自动存表格

**不用写代码，不用API，个人完全能用！**

飞书捷径就是：「当A发生时，自动做B」

我们要做的是：**当群机器人发消息时，自动把内容存入表格**

---

#### 配置飞书捷径：

1. 打开：https://jianting.feishu.cn （飞书捷径）
2. 点击「新建捷径」
3. **第一步：选择触发器**
   - 搜索：「群机器人@我时」
   - 选择你刚才创建的那个群

4. **第二步：选择执行动作**
   - 搜索：「向多维表格新增一行记录」
   - 选择你刚才创建的多维表格

5. **第三步：字段映射（最关键！）**

把机器人消息里的内容和表格列一一对应：

| 表格列 | 对应消息里的内容 |
|--------|----------------|
| 姓名 | 「客户」冒号后面的文字 |
| 电话 | 「电话」冒号后面的数字 |
| 需求面积 | 「面积」冒号后面 |
| 工位数量 | 「工位」冒号后面 |
| 隔断数量 | 「隔断」冒号后面 |
| 换房原因 | 「换房原因」冒号后面 |
| 意向区域 | 「区域」冒号后面 |
| 提交时间 | 消息发送时间 |
| 跟进状态 | 固定填「待联系」 |

✅ ✅ ✅ 完成！

就用 Vercel，30秒搞定，永久免费：
