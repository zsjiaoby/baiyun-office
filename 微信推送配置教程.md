## 🚀 配置只需要 30 秒

### 1. 拿SendKey
打开这个网址：https://sct.ftqq.com

- 用微信扫码登录
- 就能看到你的 SendKey，长这样：
  ```
  SCT123456Txxxxxxxxxxxxxxxxxxxxxxxxxxxx
  ```

### 2. 填到代码里
打开 `server.js`，把第一行改成你刚拿到的SendKey：

```javascript
const SERVERCHAN_SENDKEY = 'SCT123456Txxxxxxxxxxxxxxxxxxxxxxxxxxxx';
```

✅ **就这两步，完成了！**

## 🎯 推荐个人配置

只要这一个就够了！其他都可以不用开：

```javascript
// server.js 只需要改这一行，其他都不动
const SERVERCHAN_SENDKEY = '你的SendKey';
```

部署上线，然后你就可以：
- 微信发链接给客户
- 客户填写提交
- ✨ 你的微信叮咚！收到提醒 ✨
- 点一下通知就能看到完整客户信息
- 直接拨电话跟进！
