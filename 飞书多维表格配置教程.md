## 🚀 5分钟配置完成

---

### 第一步：创建飞书多维表格

1. **新建多维表格**
   - 打开飞书 → 新建 → 多维表格
   - 命名：「白云办公室客户线索表」

2. **添加这些列（重要！列名必须完全一致）**：

| 列名 | 字段类型 |
|------|----------|
| 姓名 | 单行文本 |
| 电话 | 单行文本 |
| 需求面积 | 单行文本 |
| 工位数量 | 单行文本 |
| 隔断数量 | 单行文本 |
| 意向区域 | 单行文本 |
| 提交时间 | 单行文本 |
| 跟进状态 | **单选** |

3. **跟进状态的选项**：
   - 待联系（蓝色）
   - 已联系（黄色）
   - 已带看（绿色）
   - 已成交（红色）
   - 已放弃（灰色）

---

### 第二步：获取表格的两个ID

打开你刚创建的多维表格，看浏览器地址栏：

```
https://bytedance.feishu.cn/base/【这一段就是app_token】?table=【这一段就是table_id】&view=vewxxxx
```

✅ 例子：
```
链接：https://bytedance.feishu.cn/base/FAbMbOnRIsaPUSsdf23c9BkjnNh?table=tbluR345DFg&view=vewhaKqWeR

FEISHU_BITABLE_APP_TOKEN = "FAbMbOnRIsaPUSsdf23c9BkjnNh"
FEISHU_TABLE_ID = "tbluR345DFg"
```

把这两个值填到 `server.js` 对应位置。

---

### 第三步：创建飞书自建应用

1. 打开 **飞书开放平台**：https://open.feishu.cn/
2. 点击 **创建企业自建应用**
   - 应用名称：「办公室客户同步」
   - 描述：自动同步表单客户到多维表格
3. 创建成功后，你就能看到：
   ```
   App ID: cli_xxxxxxxxxx
   App Secret: xxxxxxxxxxxxxxxxxxxx
   ```
   ✅ 把这两个也填到 `server.js`

---

### 第四步：给应用开通权限

这是最关键的一步！

在飞书开放平台，找到你的应用：

1. **添加权限**
   - 左边菜单 → 权限管理 → 权限列表
   - 搜索并勾选这两个权限：
     ```
     ✅ 云文档 - 多维表格
     「获取多维表格数据」
     「编辑多维表格数据」
     ```
   - 点击「批量申请」

2. **发布版本**
   - 左边菜单 → 应用发布 → 创建版本
   - 版本号：1.0.0
   - 点击「申请发布」
   - ✅ 用飞书管理员账号通过审批

3. **把应用添加到多维表格**
   - 回到多维表格 → 右上角「...」→ 文档设置
   - 高级设置 → 添加协作者
   - 搜索你的应用名字「办公室客户同步」
   - 设置权限为「可编辑」

---

### 第五步：配置群机器人（消息提醒）

1. 打开飞书群 → 右上角「...」→ 群设置 → 机器人
2. 添加机器人 → 自定义机器人 → 名字「新客户提醒」
3. 复制 webhook 地址填到 `server.js`

---

## ✅ 最后检查你的 server.js 配置

```javascript
// 这5项都填对就大功告成了！
const FEISHU_WEBHOOK = 'https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxx';
const FEISHU_APP_ID = 'cli_xxxxxxxxxx';
const FEISHU_APP_SECRET = 'xxxxxxxxxxxx';
const FEISHU_BITABLE_APP_TOKEN = 'xxxxxxxxxxxxxx';
const FEISHU_TABLE_ID = 'tblxxxxxxxx';
```


## ❓ 常见问题

**Q：数据没写入表格？**
- A：检查应用是不是添加为表格协作者了（这步90%的人忘）

**Q：提示权限不足？**
- A：开放平台的多维表格编辑权限申请了吗？管理员审批通过了吗？

**Q：只想用不想这么复杂？**
- A：也可以直接用一个简单方案：提交的时候推送到你的个人微信/企业微信，不用存表格也行。
